- Folders
	- data\original\
		Folder of images that need to be filled with images in data\piece.
		(Recommend to make them with same size)
	- data\piece\
		Folder of images that will fill images in data\original.
		(Recommend to make them with same size)
	- data\pref.txt
		Preference.
	- result\
		Folder of results.
- About pref.txt
	- 1st line
		Width of result image(px).
	- 2nd line
		Height of result image(px).
	- 3rd line
		Width of piece image(px).
	- 4th line
		Height of piece image(px).
